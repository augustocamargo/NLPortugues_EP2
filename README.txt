1. Colocar no dir src os arquivos B2W-Reviews01.csv e cbow_s50.txt
2. Executar o Script ep2.py (ele faz tudo: pre-processamento, treinamento e teste)
3. Os resultados serão gerados no dir src mesmo.
4. No dir outputs está os resultados do grid search de 72 experimentos:
    charts: gráficos de acurácia e loss. Histograma de palabras.
    histories: serialização do objeto history nos arquivos .hist e tb export no formato .csv
    resultados: acurácia do melhor modelo salvo aplicado no corpo de teste
